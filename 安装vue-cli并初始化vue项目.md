# 1. 全局安装vue-cli
~~~
npm install -g vue-cli

-g 表示全局安装
~~~
![](images/screenshot_1528102850652.png)

# 2 初始化vue项目

vue-demo是项目名称(安装时可修改)
~~~
vue init webpack vue-demo
~~~
注意这一步骤
